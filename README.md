# backend-ui

## Getting Started

- npm install
- 本地调试 npm run dev
- 生成 test 版本 npm run build:test
- 生成 release 版本 npm run build:prod

###风格指南

https://cn.vuejs.org/v2/style-guide/

### Views

在 views 文件下，代表路由的.vue 文件都使用横线连接 (kebab-case)，
代表路由的文件夹也是使用同样的规则。
它需要和 component 进行区分,所以路由对应的 view 应该要保持统一

例子：back-to-top.vue
